Code to reproduce the results from, and impelement the functionality described in, the paper "A Bayesian error model for synthesis and sequencing of oligonucleotides", Marrs, FW, Gratz, D, and Erkkila, TH. This code was released by DOE/NNSA with reference number O4821. 

# functions.R
All of the functionality is contained in the functions in this file. 

# vignette.R
This code illustrates many of the functionality in functions.R.
